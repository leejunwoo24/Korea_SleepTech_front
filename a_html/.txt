[회원]
 홈페이지 접속에 접속한다.

 회원은 회웍 가입 유형 중 "회원"을 선택하여 가입을 진행한다.

 회원가입은 다음과 같이 진행된다..
    1-1. 회원가입
        1. 개인 정보 입력
        개인 정보는 아이디, 비밀번호, 비밀번호 확인, 이름, 생년월일, 성별, 휴대폰 번호, 이메일, 주소를 입력하여 진행한다.

        - 아이디는 영문/숫자를 포함한   5 ~ 12자로 입력한다 
        - 비밀번호는 영문/숫자/특수문자를 포함한 8 ~ 15자로 입력한다.
        - 비밀번호 확인은 비밀번호와 비밀번호 확인과 일치해야 한다.
        - 아이디, 휴대폰 번호, 이메일은 이미 가입된 회원과 중복될 수 없다.
          
           - 위의 내용을 다 기입했으면 폼 설문조사로 넘어간다.
        
        2. 폼 설문조사
        회원가입에 개인정보를 작성하고 난 후 폼 설문조사가 진행이 된다.
        폼 설문조사 내용은 다음과 같다.
            - 연령(선택) (1) 18-29   (2) 30-39   (3) 40-49   (4) 50이상
            - 체형(선택) (1) 마름     (2) 보통     (3) 살집있음
            - 목표(선택) (1) 체중감량  (2) 근육량 증진  (3) 퍼포먼스 향상
            - 체지방 수준 선택(범위 설정) 5-9% 10-14% 15-19% 20-24% 25-29% 30-34% 35-39% 40%이상
            - 개선시키고 싶은 부위(선택, 복수선택 가능) (1) 가슴 (2) 팔  (3) 배  (4) 다리 (5) 따로 없음
            - 따르고 있는 식단 유뮤(선택) (1) 채식 (2) 비건 (3) 케토 (4) 지중해식 (5) 카니보어 (6) 해당사항 없음
            - 단 음식이나 음료를 얼만나 자주 섭취하시나요?(선택) (1) 자주 먹지는 않는다 (2) 주 3~5회 정도 (3) 매일
            - 수분 섭취 유뮤(선택)(1컵 = 250ml) (1) 커피 또는 차만 (2) 2잔 미만 (3) 2~6잔  (4) 7~10잔 (5) 10잔 이상
            - 키(cm) (input태그 type= "text"로 입력)
            - 현재 체중(kg) (input태그 type= "text"로 입력) , 목표 체중(kg) (input태그 type= "text"로 입력)
            - 체력수준(범위 설정) 1 ~ 10
            - 이전에 운동을 시도하면서 겪었던(선택) (1) 동기 부족 (2) 많은 시도를 해봤지만 효과를 못 봄 (3) 운동이 너무 힘들었음 (4) 뚜렷한 계획이 없음 (5) 잘못된 코칭 (6) 해당 사항 없음 
            - 한라운드에 팔굽혀펴기를 얼마나 할 수 있나요 (선택) (1) 10개 미만 (2) 10~20개 (3) 21~30개 (4) 30개 이상
            - 한라운드에 턱걸이를 몇 번이나 할 수 있나요 (선택) (1) 0개 (2) 5개 미만 (3) 5개 ~10개 (4) 10개 이상
            - 평소에 운동은 일주에 몇 번 하시나요(선택) (1) 전혀하지 않음 (2) 주 1~2회 (3) 주 3회 (4) 주 3회 이상
            - 하루에 운동에 투자할 수 있는 시간은 어떻게 되시나요(선택) (1) 30분 (2) 45분 (3) 1시간 (4) 트레이너의 결정에 맡긴다.

        3. 회원가입 완료


    2-2. 로그인
      : 아이디 & 비밀번호 입력
          - 아이디 비밀번호 둘 다 틀릴 시 >> 출력: 아이디와 비밀번호가 일치하지 않습니다.
          - 아이디만 틀렸을 시 >> 출력: 아이디가 일치하지 않습니다.  
          - 비밀번호만 틀렸을시 >> 출력: 비밀번호가 일치하지 않습니다.
      
      : 아이디, 비밀번호 찾기
          - 아이디 찾기
              1. 이름을 입력
              2. 생년월일을 입력
              3. 이메일을 입력
            
          - 비밀번호 찾기
              1. 아이디를 입력
              2. 이름을 입력
              3. 생년월일을 입력
              4. 이메일로 인증번호가 전송
                 -> 전송된 인증번호를 확인한 후 인증번호를 입력

              5. 비밀번호 찾기가 완료되면, 새 비밀번호를 입력하여 비밀번호를 재설정한다.
                 - 새 비밀번호를 입력한다.
                 - 비밀번호 확인을 입력한다
                    : 비밀번호 확인은 새 비밀번호와 비밀번호 확인 입력값이 일치해야함

      
      : 로그인 완료
          - 로그인 완료여부 모달창으로 띄운 후 확인버튼 클릭시 회원 계정의 메인 홈 화면으로 이동



3. 메인 홈 화면
  구독 결제를 한 회원과 구독 결제를 하지 않은 회원이 나뉘어짐.

  
  구독 결제를 하지 않은 회원(유령 회원)
  
  유령회원이 이용할 수 있는 기능은 다음과 같음

  - 마이페이지 기능
      1. 계정관리
        : 계정 정보를 수정(아이디, 비밀번호, 주소), 계정 탈퇴 기능, 구독 회원 전환 여부(클릭시 구독 결제 창으로 이동)
      
      2. 알림확인
        : 커뮤니티 확인
        : 리뷰 확인
        이 2가지가 리스트 형식으로 알림확인에 들어가면 나타난다.
      
      3. 구매한 상품 관리
      구매한 오프라인 상품(pt 1회 체험권 할인, PT샵 이용권 할인 등등)을 관리하는 페이지
      : 구매한 상품을 확인할 수 있다.(상품 클릭시 상품 상세페이지로 랜딩)
      : 결제내역을 확인할 수 있다.



  - 커뮤니티 기능
    글id, 글 제목, 글 내용, 조회수, 공감수, 댓글 수, 작성날짜가 
    나타남 

    1.글 조회 시 
     : 사람들이 작성한 글 리스트를 확인
     : 읽고 싶은 글을 리스트에서 클릭해서 들어감
     : 읽은 글에 댓글과 공감표시 가능
     : 댓글에 공감표시와 대댓글을 작성할 수 있다.
     : 자신이 작성한 댓글을 수정하거나 삭제할 수 있다.
        -> 댓글 수정시 수정이 되었다는 표시가 뜸뜸

    2. 글 작성 시
     : 자신이 작성하고 싶은 글을 작성해 커뮤니티에 올릴 수 있다.
     : 제목, 내용(글과 사진)을 작성한 후 글을 등록
     : 수정 기능을 통해 제목 혹은 내용을 수정할 수 있다.
     : 자신이 작성한 글 중 삭제하고 싶은 글을 삭제할 수 있다.
     

  - 오프라인 쇼핑몰
    : 사이트와 제휴하고 있는 헬스장 중 회원과의 주소와 가까운 헬스장의  pt권을 쇼핑몰 형식으로 판매
    
    : 판매하는 상품은 할인된 PT 1회 체험권 , 할인된 PT샵 이용권 등등이 판매된다. 

    : 판매되는 상품의 리뷰를 확인할 수도 있다.
      - 리뷰는 커뮤니티의 형식과 유사하게 리스트 형식으로 나타난다.
      - 리뷰에 공감표시, 댓글을 작성할 수 있다.
      - 리뷰에 달린 댓글에 공감표시나 대댓글을 작성할 수 있다.
      - 작성한 댓글은 수정, 삭제가 가능하다.


  - 리뷰 작성 기능
    : 자신이 구매한 상품(ex) PT권)에 대해 리뷰를 작성할 수 있다.
     - 글 id, 글 제목, 글 내용(텍스트, 사진), 공감수, 댓글, 별점, 작성날짜가 리뷰 리스트에 표시된다.
     - 작성한 리뷰를 수정하거나 삭제할 수 있다.


  - QnA?

    





구독 결제 후
  
  구독 결제를 한 회원(구독 회원)

  - 마이페이지 기능
      1. 계정관리
        : 계정 정보를 수정(아이디, 비밀번호, 주소), 계정 탈퇴 기능
      
      2. 알림확인
        : 커뮤니티 확인
        : 리뷰 확인
        : 쪽지 확인
        이 3가지가 리스트 형식으로 알림확인에 들어가면 나타난다.
      
      3. 구독 관리
      구독에 관한 정보와 기능이 나타난다
      - 구독 시작일자, 구독 종료일자, 가격, 구독해지 기능, 구독 연장 기능 


  - 커뮤니티 기능
    글id, 글 제목, 글 내용, 조회수, 공감수, 댓글 수, 작성날짜가 
    나타남 

    1.글 조회 시 
     : 사람들이 작성한 글 리스트를 확인
     : 읽고 싶은 글을 리스트에서 클릭해서 들어감
     : 읽은 글에 댓글과 공감표시 가능
     : 댓글에 공감표시와 대댓글을 작성할 수 있다.
     : 자신이 작성한 댓글을 수정하거나 삭제할 수 있다.
        -> 댓글 수정시 수정이 되었다는 표시가 뜸뜸

    2. 글 작성 시
     : 자신이 작성하고 싶은 글을 작성해 커뮤니티에 올릴 수 있다.
     : 제목, 내용(글과 사진)을 작성한 후 글을 등록
     : 수정 기능을 통해 제목 혹은 내용을 수정할 수 있다.
     : 자신이 작성한 글 중 삭제하고 싶은 글을 삭제할 수 있다.


  - 트레이너와 매칭 기능
     매칭 전
       - 랜덤 매칭
          : 버튼 클릭시 무작위로 근처 지역에 있는 트레이너와 랜덤으로 매칭
          : 랜덤으로 매칭된 트레이너에 대한 정보를 확인할 수 있다
          : 트레이너와 채팅 문의 버튼

        - 직접 선택
          : 소개, 리뷰확인, 별점, 댓글, 트레이너 경럭, 트레이너 스케줄 등등을 회원이 확인하고, 자신에게 맞는 트레이너를 선택
          : 트레이너와의 채팅 문의 버튼
  
        

    매칭 후
         매칭이 되면 회원이 회원가입을 하면서 작성했던 문답 폼이 트레이너에게 전달
        
         회원의 운동이 끝난 후 트레이너의 섬세한 피드백을 위한 문답 폼을 제공받는다.

         트레이너와 회원은 쪽지로 상호작용
         
         회원은 트레이너에게 식단, 운동루틴, 등등을 제공 받음


- 리뷰 작성 기능
    : 자신과 매칭된 트레이너에 대해 리뷰를 작성할 수 있다.
     - 글 id, 글 제목, 글 내용(텍스트, 사진), 공감수, 댓글, 별점, 작성날짜가 리뷰 리스트에 표시된다.
     - 작성한 리뷰를 수정하거나 삭제할 수 있다.
